crooner
===

Mostly good enough presentation software.

# features:

- textile formatting
- code syntax highlighting
- simple navigation

## quickly built with 

- http://sinatra.github.com
- http://datamapper.org
- http://code.google.com/p/syntaxhighlighter


## to start the application run

	ruby crooner.rb
